HTML Elements

<html>
  • Root of an HTML document
  • Defines html 5
  • Define the language of the document
<head>
  • The next tag after head
  • Defines meta-data
  • We define the title of the wepage in the head tag
<h1>
  • Defines a header
  • Makes the text bigger
  • h1 is the biggerest, followed by h2, h3, h4, h5 and h6
<div>
  • Generic flow container
  • Used to divide styling choices for CSS
  • Seperated by classes and accessed through the style sheet

HTML Character Entities

<
  • less than character
  • represented as &lt; in code
>
  • greater than character
  • represented as &gt; in code
&
  • ampersand
  • represented as &amp; in code

CSS Selectors

.class
  • styles all elements with the "class" name
  • example: <div class="main_boody">
element
  • styles all elements of the element name
  • example: p
  • selects all <p> elements and styles them
element01 element2
  • styles all elements02 inside elements01
  • example: div h1
  • selects all <h1> elements inside <div>
#id
  • styles all elements inside a tag with the id
  • example: #styling_choice
  • <h1 id="styling_choice"> ... </h1>

CSS Properties

background-color
  • changes the background color of the element
  • example: blue, red, #d48c8c
margin-right
  • changes the margin of the element
  • can also be margin-left, margin-bottom, margin-top
  • example: 15px, 8%
display
  • tells how an HTML element should be displayed
  • example: none, inline, block, inline-block
background-image
  • Duplicates the image to the background
  • By default, a large image is not scaled down to fit into it's container
background-repeat
  • no-repeat - stop the background from repeating altogether
  • repeat-x - repeat horizontally
  • repeat-y - repeat vertically
  • repeat - the default; repeat in both directions
background-size
  • Use this property to scale a background image properly
  • cover The image will be just large enough so that it completely covers the box area while still retaining it's aspect ratio. Parts of the image are likely to end up outside the box, and cut off.
  • contain The browser will make the image fit inside the box. If the aspect ratio is different, you might have blank spaces on the top or bottom.
background-position
  • Specifies where the image will start. This uses a coordinate system in which the top-left-hand corner of the box is (0,0)
  • Takes in two values, a horizontal value and a vertical value.
inline-size
  • Mozilla Docs
  • Inline size determines the horizontal or vertical size of an element's block, depending on its writing mode
  • Kind of like width and height
