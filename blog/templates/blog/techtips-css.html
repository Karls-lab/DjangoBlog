HTML Elements

<html>
  • Root of an HTML document
  • Defines html 5
  • Define the language of the document
<head>
  • The next tag after head
  • Defines meta-data
  • We define the title of the wepage in the head tag
<h1>
  • Defines a header
  • Makes the text bigger
  • h1 is the biggerest, followed by h2, h3, h4, h5 and h6
<div>
  • Generic flow container
  • Used to divide styling choices for CSS
  • Seperated by classes and accessed through the style sheet

HTML Character Entities

<
  • less than character
  • represented as &lt; in code
>
  • greater than character
  • represented as &gt; in code
&
  • ampersand
  • represented as &amp; in code

CSS Selectors

.class
  • styles all elements with the "class" name
  • example: <div class="main_boody">
element
  • styles all elements of the element name
  • example: p
  • selects all <p> elements and styles them
element01 element2
  • styles all elements02 inside elements01
  • example: div h1
  • selects all <h1> elements inside <div>
#id
  • styles all elements inside a tag with the id
  • example: #styling_choice
  • <h1 id="styling_choice"> ... </h1>

CSS Properties

background-color
  • changes the background color of the element
  • example: blue, red, #d48c8c
margin-right
  • changes the margin of the element
  • can also be margin-left, margin-bottom, margin-top
  • example: 15px, 8%
display
  • tells how an HTML element should be displayed
  • example: none, inline, block, inline-block
